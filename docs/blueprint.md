# **App Name**: SmartPig Lite

## Core Features:

- KPI Dashboard: Interactive dashboard providing a quick overview of key metrics (KPIs) like born per sow/year and weaning to estrus interval.
- Streamlined data capture: Simple data entry forms with date pickers, dropdowns, and number inputs. Validation to ensure proper inputs.
- User Authentication: Users can log in via email/password combination.
- Filter pigs: Filter pigs and view historic and current pig information on pig overview page
- Adaptable UI design: Responsive design that adapts to different screen sizes.
- Feeding Suggestion Tool: Based on data inputs of all parameters about feeding, make intelligent suggestions about types of feeding and food to avoid illness and increase harvest.

## Style Guidelines:

- Primary color: Vibrant coral (#FF7F50), evokes vitality and warmth, nodding to the life-centric focus of pig farming. The orange/red color suggests growth, birth, health, and productiveness. It stands out effectively against the chosen background.
- Background color: Pale peach (#FAEBD7), creating a light, airy, and clean backdrop for the interface. It will ensure readability and provide a calm visual experience. This choice, while subtle, evokes a sense of nature and agricultural environments, harmonizing with the subject matter.
- Accent color: Muted terracotta (#E07A5F), adding a touch of earthiness and highlighting key interactive elements such as buttons, selections, and notifications. The use of earth tones can further echo the nature-friendly feel of the application, without directly pointing towards it. 
- Body and headline font: 'Inter', a grotesque-style sans-serif with a modern, machined, objective, neutral look. Note: currently only Google Fonts are supported.
- Use clear, outline-style icons related to pig farming activities.
- Clean and modern layout with ample white space for readability.
- Subtle transitions and animations to enhance user experience.
